# Point Cloud Hidden Point Removal using Camera Trajectory

Implementation of a frame-wise Hidden Point Removal (HPR) pipeline for dynamic point clouds based on the viewport-dependent optimization idea presented in the ISM 2024 paper.

The project processes point cloud sequences together with a camera trajectory and removes points that are not visible from the user's viewpoint.

## Method

For every frame:

```text
P_i -> HPR -> P'_i
```

The ISM24 implementation uses three viewpoints:

```text
C1 = user viewpoint

C2 = C1 + [deltaX, 0, 0]

C3 = C1 - [deltaX, 0, 0]
```

Visible points are computed from all viewpoints:

```text
Visible =
HPR(P,C1)
в€Є HPR(P,C2)
в€Є HPR(P,C3)
```

The final point cloud is generated by taking the union of all visible points.

## Parameters

### Multi-view offset

```python
DELTA_X = 500
```

### HPR radius factor

```python
HPR_RADIUS_FACTOR = 10000
```

Higher values generally preserve more visible points.

### Maximum frames

```python
MAX_FRAMES = 300
```

or

```python
MAX_FRAMES = None
```

for all frames.
